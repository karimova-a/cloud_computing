# JTBD Interviewer Agent

A Gemini-based CLI agent that interviews users using the Jobs to be Done (JTBD) framework and generates a structured PRD.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Set your Google API key:
   ```bash
   export GOOGLE_API_KEY="your_api_key_here"
   ```

## Usage

Run the agent:
```bash
python main.py
```
